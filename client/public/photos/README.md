# 📸 Папка для фотографий

Положите сюда ваши фото с именами:
- `photo1.jpg`
- `photo2.jpg`
- `photo3.jpg`
- `photo4.jpg`
- `photo5.jpg`
- `photo6.jpg`

Если фото отсутствуют — сайт автоматически покажет красивые плейсхолдеры с Unsplash.

Рекомендуемый размер: **800×1000 px** (вертикальная ориентация 4:5).
Поддерживаемые форматы: jpg, png, webp.
